### 闭包是什么？
- 闭包是种特殊的对象
- 它有两部分组成，执行上下文(比如：A),以及在该执行上下文中创建的函数(比如：B)，当B执行时，如果访问了A中的变量，那么就产生闭包

```javascript
function A(){
    let a=10;
    return function B(){
        console.log(a);
    }
}

const B=A();
B();//B执行，全局作用域下没有a的定义，但是函数闭包，能够把定义函数的时候作用域一起记住
```

### 闭包形成的条件
- 函数嵌套
- 内部函数引用外部函数的变量

### 闭包的问题
> JS拥有自动的垃圾回收机制，当一个变量，在内存中失去引用时，垃圾回收会在适当的时机回收它，释放内存。但是用于闭包的存在，变量还是在被引用内存不能被释放
